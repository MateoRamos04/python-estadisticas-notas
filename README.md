# Python - Estadísticas de Notas

Pequeño programa en Python que permite:

- Cargar notas (1 a 10)
- Calcular cantidad de aprobados
- Calcular cantidad de desaprobados
- Calcular promedio
- Obtener nota mínima
- Obtener nota máxima


## Tecnologías

- Python
- Git
- Linux
